# 1. INTRODUCCIÓN

La presente Guía de Estilos UI establece los lineamientos visuales oficiales del portal Matemática Digital.

Su propósito es garantizar uniformidad visual, consistencia de componentes, accesibilidad, mantenibilidad y coherencia gráfica durante todas las etapas de diseño y desarrollo.

Este documento constituye la referencia oficial para la construcción de Mockups, Prototipos, HTML5, CSS3 y componentes visuales del proyecto.

---

# 2. OBJETIVOS

## Objetivo General

Definir los lineamientos visuales oficiales que regirán el diseño de la interfaz del portal Matemática Digital.

## Objetivos Específicos

* Definir la identidad visual institucional.
* Documentar la paleta de colores oficial.
* Establecer tipografías y jerarquías visuales.
* Normalizar componentes UI.
* Facilitar la construcción de Mockups.
* Mantener consistencia durante el desarrollo.

---

# 3. ALCANCE

# 4. PRINCIPIOS DE DISEÑO

## Claridad

La información debe ser comprensible y fácilmente identificable por el usuario.

---

## Consistencia

Todos los componentes deberán mantener comportamiento y apariencia uniforme.

---

## Accesibilidad

La interfaz deberá facilitar la navegación para la mayor cantidad posible de usuarios.

---

## Simplicidad

Se priorizará la reducción de elementos visuales innecesarios.

---

## Escalabilidad

La interfaz deberá permitir futuras ampliaciones del portal sin afectar la coherencia visual.

# 7. TIPOGRAFÍA

## Objetivo

Establecer un sistema tipográfico uniforme que garantice legibilidad, accesibilidad y coherencia visual en todas las páginas del portal Matemática Digital.

---

## Tipografía Principal

### Poppins

La tipografía principal del proyecto será Poppins.

Aplicaciones:

* Títulos principales
* Subtítulos
* Menús de navegación
* Botones
* Encabezados de tarjetas
* Indicadores estadísticos

Características:

* Moderna
* Profesional
* Geométrica
* Fácil lectura en pantalla

Ejemplo:

```text
Matemática Digital
```

---

## Tipografía Secundaria

### Roboto

La tipografía secundaria será Roboto.

Aplicaciones:

* Párrafos
* Artículos
* Recursos educativos
* Formularios
* Tablas
* Textos descriptivos

Características:

* Excelente legibilidad
* Amplia compatibilidad
* Optimizada para dispositivos digitales

Ejemplo:

```text
Recursos, estrategias e innovación para la enseñanza de las matemáticas.
```

---

## Tipografía Monoespaciada

### Roboto Mono

Aplicaciones:

* Fragmentos de código
* Ejemplos HTML
* Ejemplos CSS
* Configuraciones técnicas

Ejemplo:

```css

```
