# 1. INTRODUCCIÓN

El presente documento describe la estructura visual de baja fidelidad (Wireframes) correspondiente al portal web Matemática Digital.

Los wireframes representan la organización de los componentes, secciones y áreas funcionales de cada página antes de la aplicación de estilos visuales, colores, tipografías e imágenes definitivas.

Este documento mantiene coherencia con los requerimientos funcionales, la arquitectura de información, el sitemap y el inventario de contenidos previamente aprobados.

---

# 2. OBJETIVOS

## Objetivo General

Definir la estructura visual y distribución de contenidos de las páginas que conforman el portal Matemática Digital.

## Objetivos Específicos

* Establecer la organización visual de cada página.
* Representar la distribución de componentes y contenidos.
* Facilitar el proceso de diseño de interfaces.
* Servir como referencia para la elaboración de mockups.
* Mantener la trazabilidad con el Inventario de Contenidos.

---

# 3. ALCANCE

Este documento comprende los wireframes de las siguientes páginas:

* Inicio
* Matemática y Educación
* Estrategias de Enseñanza
* Herramientas Digitales
* Recursos Educativos
* Blog
* Nosotros
* Contacto

# 4. RELACIÓN CON LA ARQUITECTURA DE INFORMACIÓN

Relación:

Arquitectura → Sitemap → Inventario → Wireframe

---

# 5. CONVENCIONES DE WIREFRAMES

Los wireframes utilizan:

* Bloques estructurales.
* Delimitadores de secciones.
* Etiquetas descriptivas.
* Marcadores de imágenes.
* Representación de formularios.
* Indicadores de CTA.

Los wireframes no incluyen:

* Colores.
* Tipografías.
* Iconografía definitiva.
* Imágenes finales.
* Animaciones.
